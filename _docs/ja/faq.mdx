## 一般

#### Q. HTML や CSS セレクターの知識がなくても拡張機能を使うことはできますか？

特別な知識がなくてもキーワードブロック機能を使うことができます。  
キーワードブロックを自分好みの動作にカスタマイズしたい時だけ、HTML や CSS セレクターの知識が必要になることがあります。

---

#### Q. モバイル版を購入すれば iOS と iPadOS の両方で使えるようになりますか？

はい、両方で使えるようになります。

---

#### Q. モバイル版は月額課金ですか？

永久ライセンスでの提供となります。買い切り型のため、月々の料金が発生することはありません。

---

#### Q. Android 版は存在しますか？

Android 版の Chrome ではブラウザ拡張機能を利用できないため、Android 版はサポートしていません。

---

#### Q. Dropbox 以外のクラウドと連携できますか？

現在は Dropbox のみサポートしております。

## アプリの機能について

#### Q. 要素やテキストを隠す絵文字を変更したい

オプションページの「絵文字」でお好きな絵文字を設定することができます。

---

#### Q. 特定のサイトで拡張機能をオフにしたい

拡張機能をオフにしたいサイトで EmoGuard を開き、電源ボタンをクリックしてオフにしてください。

---

#### Q. キーワードリストを常に表示したい

オプションページの「キーワードリストを常に表示」を有効にしてください。

---

#### Q. サイト別にキーワードを設定できますか？

サイト別にキーワードを設定することはできません。

---

#### Q. 「おすすめ CSS セレクター」とはなんですか？

拡張機能のインストール後すぐにキーワードブロック機能が使えるように、あらかじめセットアップされた CSS セレクターです。  
一般的なサイトや有名な主要サイトで動作するように調整されています。  
おすすめ CSS セレクターの利用は任意であり、利用する場合でも設定値を自由にカスタマイズすることができます。

---

#### Q. CSS セレクターが削除できません

おすすめ CSS セレクターは削除できない仕様です。  
おすすめ CSS セレクターを無効化したい場合、個別に目のアイコンをクリックして「表示」状態にするか、オプションページの「おすすめ CSS セレクター 設定」から該当のものを無効化してください。

---

#### Q. CSS セレクターの隠蔽モードとはなんですか？

隠蔽モードは「ターゲットの隠し方」を指定するための項目です。  
キーワードが一致したとき、要素とテキストのどちらを隠すのか選択することができます。

---

#### Q. CSS セレクターの検索モードとはなんですか？

検索モードは「どのようにキーワードを検索するか」を指定するための項目です。

- シャローサーチ: CSS セレクターに一致した**要素の直下の**テキストからキーワードを検索
- ディープサーチ: CSS セレクターに一致した**要素とその子孫要素の**テキストからキーワードを検索

---

#### Q. 検索モードの「ディープサーチ」が表示されず選択できません

ディープサーチは隠蔽モードが `キーワードと一致したテキストを隠す` に設定されているときのみ選択することができます。

---

#### Q. ○○○ アイコンは何を意味していますか？

基本的に拡張機能のアイコンやボタンは、数秒間マウスオーバーするとツールチップが表示されてヒントを読むことができます。  
モバイル版の場合は長押しで同様にツールチップが表示されます。

---

#### Q. ブロックしたい要素やテキストを一瞬でも表示したくない

オプションページの「キーワードブロック速度」スライダーを `速い` 方に寄せるように調整してください。  
要素やテキストが非表示になるまでの時間は短くなりますが、動作が重くなる可能性があります。

---

#### Q. Web ページの読み込みが遅い・動作が重い

オプションページの「キーワードブロック速度」スライダーを `遅い` 方に寄せるように調整してください。  
要素やテキストが非表示になるまでの時間は伸びてしまいますが、動作が軽くなる可能性があります。

---

#### Q. Web サイトの表示がおかしくなってしまいました

以下の回避策をお試しください。

- CSS セレクターの設定を変更する
- 該当の CSS セレクターを削除する
- おすすめ CSS セレクターをリセットする
- そのサイトでは拡張機能をオフにする

---

#### Q. CSS セレクターやキーワードを削除しても Web ページ上の絵文字が消えないことがあります

EmoGuard は Web ページの読み込み開始直後から動作するため、稀に Web ページで使用されているフレームワークの動作と競合してしまうことがあります。  
その際はページをリロードすることで絵文字を消すことができます。

---

#### Q. 拡張機能が動作しないサイトがあります

EmoGuard はほとんどの場所で問題なく動作するように設計されているにも関わらず、残念ながら上手く動作しないサイトが存在する可能性があります。  
Web サイトは多種多様な技術で作成されており、それら全てに対応することは技術的に困難な課題です。

EmoGuard はあらゆる全てのサイトで動作することを保証していませんが、できるだけそれらに適合するよう最大限の努力を試みます。
