## 一般

#### Q. HTML や CSS セレクターの知識がなくても拡張機能を使うことはできますか？

特別な知識がなくてもキーワードブロック機能を使うことができます。  
拡張機能の動作をカスタマイズしたい時だけ、HTML や CSS セレクターの知識が必要になることがあります。

---

#### Q. モバイル版を購入すれば iPhone と iPad の両方で使えるようになりますか？

はい、両方で使えるようになります。

---

#### Q. モバイル版は月額課金ですか？

永久ライセンスでの提供となります。買い切り型のため、月々の料金が発生することはありません。

---

#### Q. Android 版は存在しますか？

Android 版の Chrome ではブラウザ拡張機能を利用できないため、Android 版はサポートしていません。

---

#### Q. Dropbox 以外のクラウドサービスと連携できますか？

現在は Dropbox のみサポートしています。

## アプリの機能について

#### Q. 拡張機能をカスタマイズしたい

ポップアップ画面右上の「⋮」からカスタマイズを有効にすることができます。

---

#### Q. 絵文字を変更したい

オプションページの「絵文字」セクションでお好きな絵文字を設定することができます。

---

#### Q. 特定のサイトで拡張機能をオフにしたい

1. 拡張機能をオフにしたいサイトで EmoGuard を開きます。
2. ポップアップ画面右上の「⋮」からカスタマイズを有効にします。
3. 電源ボタンをクリックして拡張機能をオフにしてください。

---

#### Q. キーワードリストを常に表示したい

オプションページの「キーワードリストを常に表示」を有効にしてください。

---

#### Q. サイト別にキーワードを設定できますか？

サイト別にキーワードを設定することはできません。

---

#### Q. 「デフォルト CSS セレクター」とはなんですか？

拡張機能のインストール後すぐにキーワードブロック機能が使えるように、あらかじめセットアップされた CSS セレクターです。  
一般的なサイトや有名なサイトで動作するように調整されています。  
デフォルト CSS セレクターの利用は任意であり、利用する場合でも設定値を自由にカスタマイズすることができます。

---

#### Q. CSS セレクターが削除できません

デフォルト CSS セレクターは削除できない仕様です。  
デフォルト CSS セレクターを無効化したい場合、個別に目のアイコンをクリックして「表示」状態にするか、オプションページの「デフォルト CSS セレクター 設定」から該当のものを無効化してください。

---

#### Q. CSS セレクターの隠蔽モードとはなんですか？

隠蔽モードは「テキストや要素の隠し方」を指定するための項目です。  
Web ページ上でキーワードが一致したとき、要素を丸ごと隠すのか、テキストをピンポイントで隠すのかを選ぶことができます。

---

#### Q. CSS セレクターの検索モードとはなんですか？

検索モードは「キーワードの検索方法」を指定するための項目です。

- シャローサーチ: CSS セレクターに一致した**要素の直下の**テキストからキーワードを検索します。
- ディープサーチ: CSS セレクターに一致した**要素とその子孫要素の**テキストからキーワードを検索します。

---

#### Q. 検索モードのメニューに「ディープサーチ」が表示されません

ディープサーチは、隠蔽モードが `キーワードと一致したテキストを隠す` に設定されているときのみ選択することができます。

---

#### Q. 各アイコンの意味を教えてください

拡張機能のアイコンやボタン上で数秒間マウスオーバーすると、ヒントや使い方が表示されます。  
モバイル版の場合は長押しでツールチップが表示されます。

---

#### Q. ブロックしたい要素やテキストを一瞬でも表示したくない

オプションページの「キーワードブロック速度」スライダーを `速い` 方に寄せるように調整してください。  
要素やテキストが非表示になるまでの時間は短くなりますが、動作が重くなる可能性があります。

---

#### Q. Web ページの読み込みが遅い・動作が重い

オプションページの「キーワードブロック速度」スライダーを `遅い` 方に寄せるように調整してください。  
要素やテキストが非表示になるまでの時間は伸びてしまいますが、動作が軽くなる可能性があります。

---

#### Q. Web サイトの表示がおかしくなってしまいました

以下の回避策をお試しください。

- ページをリロードする
- 該当の CSS セレクターの設定を変更する
- 該当の CSS セレクターを削除する
- デフォルト CSS セレクターをリセットする
- そのサイトでは拡張機能をオフにする

---

#### Q. CSS セレクターやキーワードを削除しても Web ページ上の絵文字が消えないことがあります

EmoGuard は Web ページの読み込み開始直後から動作するため、稀に Web ページで使用されているフレームワークと競合してしまうことがあります。  
恐れ入りますがページをリロードすることで絵文字を消すか、そのサイトでは拡張機能をオフにしてください。

---

#### Q. 拡張機能が動作しないサイトがあります

EmoGuard はほとんどの Web サイトで動作するように設計されていますが、残念ながら完璧ではありません。  
Web は多種多様な技術で作成されており、それら全てに対応することは技術的・現実的な困難を伴います。

EmoGuard はあらゆる全てのサイトで動作することを保証しておりませんが、できるだけ多くの Web サイトで動作するよう調整が続けられています。
