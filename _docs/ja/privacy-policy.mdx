EmoGuard は、EmoGuard が提供するサービス（以下「本サービス」といいます。）におけるユーザーについての個人情報を含む利用者情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

- EmoGuard は個人情報を一切収集**しません**。
- EmoGuard はユーザーが本サービスに入力または保存する情報を一切収集**しません**。
- EmoGuard は Dropbox 認証の際、Dropbox にデータを保存するために必要なトークン以外の情報を収集**しません**。

### アクセス解析ツール

- ブラウザ拡張機能ではアクセス解析ツールを使用**しません**。
- ウェブサイトでは Cloudflare Web Analytics を使用**します**。Cloudflare Web Analytics は Cookie を使用しません。また、訪問者の個人情報を収集したり、閲覧したサイトを追跡したりしません。詳しくは、[公式サイト](https://www.cloudflare.com/web-analytics/)及び[ドキュメント](https://developers.cloudflare.com/analytics/web-analytics/)を参照してください。

### プライバシーポリシーの変更手続

EmoGuard は、必要に応じて本ポリシーを変更します。但し、法令上ユーザーの同意が必要となるような本ポリシーの変更を行う場合、変更後の本ポリシーは、EmoGuard 所定の方法で変更に同意したユーザーに対してのみ適用されるものとします。なお、EmoGuard は、本ポリシーを変更する場合には、変更後の本ポリシーの施行時期及び内容を EmoGuard のウェブサイト上での表示その他の適切な方法により周知し、またはユーザーに通知します。
